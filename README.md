# Student Registration & ID Card Generation System  

This is a desktop-based Student Registration System developed using Python.The application allows users to manage student records efficiently with features like Excel import/export, PDF generation, and automated ID card creation with QR codes.

## Features
 -Add, Update, Delete student records (CRUD operations)
 -SQLite database integration
 -Input validation using Regular Expressions
 -Import student data from Excel (.xlsx)
 -Export student data to Excel
 -Export student table to PDF
 -Generate ID cards (single or bulk) in PDF format
 -QR code generation for each student ID
 -Optional photo support for ID cards

## Technologies Used
 - Python
 -Tkinter (GUI)
 -SQLite (Database)
 -Pandas (Excel Handling)
 -ReportLab (PDF Generation)
 -Regular Expressions (Validation)
